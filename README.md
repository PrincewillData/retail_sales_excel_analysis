Retail Sales Analysis (Excel)

Analyze retail sales data to understand product performance and revenue trends.

Tools Used  
Excel

Project Structure  
- data: dataset used for analysis  
- dashboard: Excel dashboard file

- Analyzed product performance and revenue trends

- Identified top-performing products by revenue

- Provided clear visualization of sales trends
